LinearReg notebook is for predicting the range of weight of a person by entered height and gender.
gym.py python file to load the website from streamlit
Trainingset is the dataset of height, weight and gender of 3000 values

Steps to run the project

Step - 1 : Enter your api key in gym.py file in groq_api_key value
Step - 2 : Run the model LinearReg in jupyter notebook
Step - 3 : Enter the command "streamlit run gym.py" in cmd.
